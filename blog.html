ブログサイト制作
BlogNote

概要

一年生後期集中授業で自分のブログを制作しました。 初めてWordPressを使い、実際にサイトを公開した作品です。

制作時期 2023年7月〜
制作期間 約3ヶ月
制作人数 個人制作
使用したもの HTML CSS JavaScript Sass Jquery WordPress
Illustrator Photoshop VisualStudioCode
URL https://hirochan.chillout.jp/blog/

この作品のコンセプトがキャンパスノートなので、手書き風のフォントとしてこちらのフォントを採用しました。 また、実装の簡単さからGoogleフォントを選びました。

デザインコンセプト

はじめにデザインの大まかな方針を文房具、ノートにしました。それからキャンパスノートをモチーフにすることに決めました。完成版ではモノトーンですが、はじめはヘッダーとフッターの背景色にキャンパスノートでイメージの強い水色にしていました。メインの背景には方眼ノートをデザインしました。 その上にさらに文豪具のイラストをいれて、可愛らしさを出しました。 見出しの下には、疑似要素で蛍光ペンで引いた風な線をつけました。

完成版

完成版のカテゴリー部分では付箋のように表示したり、プロフィール画像の上に画鋲を刺したりしてよりコンセプトに近づけました。
使用したフォントが手書き風なため、よりノートらしい印象に完成しました。

意識したところ

ヘッダーをノートに表紙のようなデザインにしていたので、　 めくるようなアニメーションにしたくて試行錯誤しました。

カテゴリー一覧の要素、 疑似要素を使って影を入れ、付箋の端が少し浮いている雰囲気を再現しました。
ここがとても可愛くできたのでお気に入りです。

作品制作を通して

とても時間がかかりましたが、自分のデザインが形になっていく楽しさを初めて感じることができました。 とくにヘッダーのアニメーションの実装は何日もかかったので思い通りに動いた瞬間の気持ちよさは忘れられません。完成の達成感だけで頑張って良かったと思えました。 また、最初のデザインにはなかったカテゴリーの付箋のデザインはアイデアを思いついた嬉しさも実装できた喜びもあって、作っていて本当に楽しく、Web制作の楽しさを体感できました。
